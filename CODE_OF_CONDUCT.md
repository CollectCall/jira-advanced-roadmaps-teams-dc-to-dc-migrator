# Code of Conduct

## Our standard

Participation in this project should be respectful, direct, and professional.

Examples of behavior that contributes to a constructive environment:

- assuming good intent
- giving technical feedback with evidence
- focusing criticism on code, docs, and decisions rather than people
- helping keep operational and security discussions responsible

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- doxxing or publication of private information
- abusive or discriminatory language
- bad-faith disruption of issues, PRs, or community spaces

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or contributions that violate this Code of Conduct.

## Reporting

Report Code of Conduct concerns to the repository maintainers through a private GitHub contact path rather than in a public issue.

All reports will be reviewed and handled as confidentially as practical.

This Code of Conduct is adapted from the Contributor Covenant, version 2.1:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
